Aura Angular is the Angular version of the *Aura Scouting* web project.  
It is a clean, lightweight starting point for managing scouting data, profiles, and media (media gallery is removed for simplicity).  

The project is structured to be modular and scalable, making it easy to add new components, modules, and features over time. It uses Angular best practices and is ready for development.

## 📦 Project Structure

- `src/` – Angular source code
- `app/` – Modules, components, and services
- `assets/` – Static resources (images, icons)
- `environments/` – Dev and production configurations

## ⚡ Requirements

- Node.js >= 18  
- Angular CLI >= 16

git clone https://github.com/Cyberdude00/Aura-angular.git
